# BlockChain-System

#FrontEnd >> HTML5&CSS3 , JavaScript, Vue.js, therefore the GUI  implemented as a JavasScript- based „server-less“ Single Page Application (SPA).

#BackEnd  >> Java Programming ,Server-side backend  implemented using Java, the OpenLiberty application server and RESTful web services and JPA to access the database.

#AI       >> Python for Machine Learning and AI , AI-based prediction for the potential price evoluttion of each
token in the nearer future, based on the underlying real estate property.  AI functions, implemented in Python as well to make use of the usual ML frameworks.
